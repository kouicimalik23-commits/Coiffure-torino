# Coiffure Torino

## Project Documentation

Welcome to the Coiffure Torino project! This project aims to provide an online platform for hair salon services, allowing users to book appointments, view services offered, and manage their profiles.

### Installation

To get started with the Coiffure Torino project:
1. Clone the repository:
   ```
   git clone https://github.com/kouicimalik23-commits/Coiffure-torino.git
   ```
2. Navigate to the project directory:
   ```
   cd Coiffure-torino
   ```
3. Install the necessary dependencies:
   ```
   npm install
   ```

### Features
- User registration and login
- Browse services offered by the salon
- Book and manage appointments
- User profile management

### Usage
To start the application, run:
```bash
npm start
```
Open your browser and navigate to `http://localhost:3000` to access the app.
